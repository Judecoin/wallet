# Judecoin CLI Wallet Documentation

## Description

This folder contains Judecoin CLI wallet documentation for Linux-Ubuntu and Windows users.

It includes guides for creating new CLI wallets and recovering existing CLI wallets.

## Documents

### Linux-Ubuntu

- [Linux-Ubuntu CLI Wallet Setup Guide](Linux-CLI-Wallet-Setup.md)
- [Linux-Ubuntu CLI Wallet Recovery Guide](Linux-CLI-Wallet-Recovery.md)

### Windows

- [Windows CLI Wallet Setup Guide](Windows-CLI-Wallet-Setup.md)
- [Windows CLI Wallet Recovery Guide](Windows-CLI-Wallet-Recovery.md)

## Notes

The CLI Wallet is intended for users who are comfortable using terminal or command-line tools.

If you prefer a graphical wallet interface, please download the [GUI Wallet](https://www.judecoin.io/downloads).

## Security Reminder

Always download Judecoin wallet software from the official Judecoin website or official GitHub source.
